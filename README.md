# 🖥️ Social Network (Tkinter)

یک برنامه ساده شبکه اجتماعی با **Python** و **Tkinter**  
این برنامه قابلیت‌های زیر را دارد:

- ثبت‌نام و ورود کاربر
- ارسال درخواست دوستی
- مشاهده پست‌های دوستان
- رابط گرافیکی ساده و مدرن با **ttk**
